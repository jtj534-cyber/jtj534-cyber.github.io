Contact Form with Supabase

This project is a contact form that allows users to send messages which are stored in a database using Supabase.

How it works

The form collects user input (name, email, and message) and sends it to a Supabase database using an API request.

What I learned

  • How to connect a front-end form to a database
  • Using APIs to send and store data
  • Handling user input in JavaScript
